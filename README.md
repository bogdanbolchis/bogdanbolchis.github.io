# bogdanbolchis.github.io

## Run on localhost

`hugo serve`

## New post 

`hugo new posts/post-name/post.md`

## Images

- add to './static'
- reference using:
	- markdown: `![Image name](/path/to/<image.ext>)`
	- html: `<img src='/<image.ext>' alt='alt'`
	
## Deploy

git commit -am "Add new post"
git push origin main
